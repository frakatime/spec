# Frakatime Specification
An open specification for tracking how much time you've spent on something

## Versions
- [V2](./V2.md) (Current)
- [V1](./V1.md)

## Implementations
List of projects that use or implement the Fraktime specification
- [ft-server-node](https://github.com/frakatime/ft-server-node) - Server implementation in NodeJS
- [ft-chrome-extension](https://github.com/frakatime/ft-chrome-extension) - Keep track of time you spend on websites
- [ft-vencord](https://github.com/frakatime/ft-vencord) - Keep track of the time you spend on Discord doing what
